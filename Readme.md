
# wordwrap

  wordwrap

## Installation

    $ component install yields/wordwrap

## API

### wordwrap(str, max, eol)

```javascript
wordwrap('Hello, World!', 5, '\n');
// > 'Hello\n, Wor\nld!'
```

### wordwrap(str, max)

```javascript
wordwrap('Hello, World!', 5);
// > 'Hello\n, Wor\nld!'
```
